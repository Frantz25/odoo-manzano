# Project Charter — Odoo Manzano

## Visión
Implementar un sistema Odoo estable, escalable y operable en vida real para Manzano.

## Alcance inicial
1. Base operativa
2. Catálogo
3. Portal cliente
4. Flujo comercial y reserva
5. Documentos
6. Comunicaciones

## Criterios de éxito
- Flujo único definido y validado
- Estados consistentes de punta a punta
- Cero confirmaciones inválidas
- Trazabilidad completa por evento

## Fuera de alcance (por ahora)
- Integraciones no críticas
- Automatizaciones avanzadas no validadas por operación real
